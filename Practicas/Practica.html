HTML

HTML (Lenguaje de marcado de Hipertexto) es un lenguaje utilizado para crear paginas web y consiste en un conjunto de etiquetas que se utilizan para definir la estructura y el contenido de la pagina , estas etiquetas marcan un inicio y un final en los elementos que contiene una pagina web como: encabezados, parrafos, imagenes, enlaces,etc.
Usando una etiqueta de apertura y otra de cierre.

<Etiqueta> </Etiqueta>
"Apertura" "Cierre"

Estructura Basica de HTML

La estructura basica en html consta de una serie de Elementos anidados se abre con la etiqueda "Html" la cual contiene dos elementos principales: el encabezado "Head" y el cuerpo "Body".

 
<html>
 <head>
 </head>
 </body>
   <h1>"Ejemplo de Titulo"</h1>
   <p>"Ejemplo de Parrafo"</p>
 </body>
</html>

Etiquetas Basicas de Html

Las Etiquetas elementos que representan una acccion especifica para que sea interpretado por el navegador algunos ejemplos son:

  • <h1> <h2> <h3> <h4> <h5> <h6> .- Permite jerarquizar el contenido, son los que se conocen como titulos y subtitulos, la especificacion puee ser de diferentes tamaños y niveles.
  • <p> .- Es una etiqueta de contenido, se utiliza para crear parrafos de texto
  • <br> .- Implementa un salto de linea dentro del texto, haciendo la funcion de la tecla "Enter" del Teclado

Etiquetas de Estilo

  • <em> .- Marca con enfasis la parte de un texto seleccionado.
  • <strong> .- Resalta el contenido de un texto con "Negrilla".
  • <code> .- Permite insertar texto que representa codigo de computadora. Permite mostrar funciones, codigo de progaramacion o variables.

Etiquetas de Creacion de Listas

  • <li> Listas no Ordenadas .-(Unorder list) Esta etiqueta sirve para crear una lista desordenada.
  • <ol> Listas Ordenadas .-(Ordered list) Esta etiqueta representa una lista ordenada de items.
  • <li>  .-(List Items) Se usa para especificar los items de la lista, ya sea en la etiqueta "ul" y "ol".
<ul>
   <li> "Ejemplo A" </li>
   <li> "Ejemplo B" </li>
 </ul>

 <ol>
   <li> "Ejemplo X" </li>
   <li> "Ejemplo Y" </li>
 </ol>
  • <dl> Listas de Deiniciones  .-(Definition list) Esta etiqueta representa una lista descriptiva.
  • <dt>   .- Especifica un termino en una descripcion o lista de definiciones.
  • <dd>   .- Provee detalles o el significado de la definicion de un termino
<dl>
   <dt> "Ejemplo de un Termino" </dt>
      <dd> "Ejemplo de Descripcion" </dd>
 </dl>

Etiquetas para insertar Imagen y Video

  • <img>  .- Esta etiqueta permite insertar imagenes en una pagina web introduciendo algunos parametros o propiedades que son:

    • "src" - Esta propiedad se usa para especificar la ruta de la imagen que se va a mostrar, citando su ubicacion y el nombre del archivo, seguido del formato que puede ser tipo Raster (.png .jpg) y Vectorial (.svg .pdf).
    "alt" - Esta propiedad se utiliza para agregar un texto alternativo para la imagen cuando no se pueda mostrar por algun motivo.
    "width" y "height" - Estas propiedades especifican el ancho y el alto de un elemento respectivamente en pixeles

  • <video>   .- Se utiliza para insertar videos en una pagina web, se aplican las propiedades "src","width" y "height" y se emplementa la propiedad:

    • "controls" -Se utiliza para mostrar los controles de reproductor de video en la pagina web, como el boton de reproduccion, pausa y volumen.

<img src="Ruta/Nombre de Imagen" width="valor px" alt="texto alternatvo" >
</img>

<video src="Ruta/Nombre del video" controls height="valor px" >
</video>

Etiqueta de Estructuracion

  • <Div>  .- Se usa para dividir el contenido de una pagina Web en secciones, no tiene ningun significado semantico en si mismo pero puede ser usado para agrupar y ordenar otros elementos usando la propiedad:

"id" - que define un identificador único el cual no debe repetirse en todo el documento

<div id= "Ejemplo de Identificador" >
   <h2> "Ejemplo" </h2>
   <p> "Ejemplo" </p>
 </div>

Hipervinculos

Los Hipervinculos en HTML son enlaces que permiten al usuario navegar ya sea de una pagina web a otra o a una seccion especifica dentro de la misma pagina web usando la etiqueta.

  • <a>  .- Que permite crear un enlace (Link) junto con la propiedad "href" que especificara la ulr del destino ya sea un archivo, una pagina web, seguido de una descripcion.
Para un Hipervinculo Externo
<a href= "Nombre de Archivo o Pagina Web" > "Descripcion"
 </a>

Para un Hipervinculo dentro de la misma Pagina Web
<a href = # "Nombre de Id" > "Descripcion"
 </a>

CSS

CSS o "Hojas de Estilo Cascada" es un lenguaje de estilo que se utiliza para dar formato y estilo a los documentos HTML. Permite controlar la presentacion visual de los elementos como el color, la tipografia, el tamaño y la posicion.

¿Como se Crea?

Existen 3 formas para poder crear estilos en un documento "HTML" las cuales son:

  1. En la propia Etiqueta.
  2. En el "Head" de la Pagina.
  3. En un Archivo Externo.

1. En la propia Etiqueta

Se crea utilizando el parametro "Style" y esta se aplica directamente a una etiqueta dentro del docuemnto HTML.

<h1 style =color:"valor" >  </h1>

2. En en Head de la pagina

Se crea utilizando la etiqueta  <style> la cual indicara la informacion de stilo que se quiere agregar, el cual debe ir dentro de la etiqueta basica  <Head> del mismo docuemnto HTML.

<head>
  <style>
        h1 {
         color:"valor";
         border:"valor";
              }
  </style>
</head>

3. En un Archivo Externo

Se crea usando la etiqueta  <link> que especifica la relacion entre el documento actual y un recurso externo. seguido de las propiedades "href" y "rel" las cuales permiten citar y especificar el enlace a una hoja externa. Las cuales deben ir dentro de la etiqueta  <Head> del mismo docuemnto HTML.

<head>
  <link href="nombre de archivo.css" rel="stylesheet">
</head>

¿Como se Usa?

Para aplicar un estilo se debe aplicar en el siguiente orden:

SELECTOR {
                          Propiedad:"valor" ;
                        }

-Selector

Un selector es un patron utilizado para seleccionar y aplicar estilos a un conjunto de elementos HTML, hay varios tipos de selectores en CSS entre ellos:

       
  • Selector de Etiqueta (h1,p,ul...)
    •    
  • Selector de "id" (#midiv, #div1...)
    •    
  • Selector de "clase" (.C1, .clase1...)

-Propiedad y Valor

Una propiedad es un atributo que se aplica a un elemento o etiqueta de un docuemnto HTML para controlar su estilo visual y que cada una debe tener un valor que define como se aplicara a la propiedad.

Propiedades para Aplicar Bordes

La propiedad utilizada es "border" seguido de un valor que puede ser en pixeles ("px") que definira el grosor de linea de borde a la vez nos permite asignar un valor de color de borde ,tambien se puede especificar un estilo de linea de borde con valores como:

  • "solid" : linea solida.
  • "dashed": puntos de linea.
  • "dotted": puntos.
  • "double": doble linea.

Otras propiedades para estilo de bordes son:
"padding" que se utiliza para establecer el espacio o margen interior entre el borde de un elmento y su contenido.
"border-raduis" que es para dar un aspecto redondeado o curvo al borde, seguido de un valor en pixeles. ("px") o porcentaje (%).

h1 {
         border:"valor en (px)" solid "valor de color";
         padding:"valor en (px) o (%)";
         border-radius:"valor en (px) o (%)";
      }

Propiedades para estilo Texto

"color" Se utiliza para dar un color al contenido seguido de un valor que puede ser en codigo # ,o directamente el texto como (red, blue, white, etc).
"background-color" Para establecer el color de fondo seguido de su respectivo valor.
"text-aling" Establece la alineacion horizontal de un texto cuyo valor puede ser:

  • "left" : alineacion a la izquierda.
  • "center" : alineacion centrada.
  • "right" : alineacion a la derecha.
  • "justify" : alineacion justificada.

"font-family" Establece una fuente o tipo de letra al contenido del selector "HTML" cuyo valor sera un nombre de fuente como por ejemplo "Helvetica", "Arial" , "courier new" ,etc.

h1 {
         color:"valor de color";
         background-color: "valor" ;
         text-aling: "valor" ;
         font-family: "nombre de fuente" ;
      }

"box-shadow" Esta propiedad permite agregar una sombra a un elemento de la pagina web, cuyos valores permiten asignar y modificar el desplazamiento tanto horizontal y vertical de la sombra tambien el desenfoque y la extension de la sombra con parametros en (px) ademas de poder dar un valor de color

p {
      box-shadow: 0px 0px 0px 0px "valor color" ;
    }

Propiedades de subrayado

Existen deiferentes tipos de subrayado que se puede aplicar a los textos usando las siguientes propiedades:
"text-decoration-line" con el valor "underline" que aplica un subrayado de linea por defecto.
"text-decoration-color" para establecer un color de subrayado seguido de su propio valor.
"text-decoration-style" permite colocar un estilo al subrayado usando valores como:

  • "dotted" : subrayado punteado.
  • "dashed" : subrayado con guiones.
  • "double" : subrayado doble linea.
  • "wavy" : subrayado ondulado.
h1 {
         text-decoration-line: underline ;
         text-decoration-color: "valor de color" ;
         text-decoration-style: "valor de un estilo" ;
      }

Pseudo-classes

Una pseudoclase en CSS es una palabra clave que se añade a los selectores y que especifica un estado especial del elemento seleccionado.
"hover" Se utiliza para seleccionar y dar estilo al pasar con el puntero del mouse por encima de un elemento.
"focus" Representa un elemento (como una entrada de formulario) que ha recibido el foco. Generalmente se activa cuando el usuario hace clic, toca un elemento o lo selecciona.

Selectores Multiples y Anidados

Selectores Multiples.- Permiten aplicar estilos a varios elementos a la vez. Para hacer esto, se utiliza una coma (,) para separar los selectores.

Selectores Anidados.- Son aquellos que se utilizan para seleccionar elementos dentro de otros elementos. Para hacer esto, se utiliza la sintaxis de un selector dentro de otro separados unicamente por la tecla espaciadora (_)

Para Selectores Multiples
h1, h2, p, ul {
                        ("Propiedades y Valor")
                      }

Para Selectores Anidados
.clase1 h1 p {
                        ("Propiedades y Valor")
                    }
